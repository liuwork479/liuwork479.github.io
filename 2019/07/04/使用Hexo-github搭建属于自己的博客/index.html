使用Hexo+github搭建属于自己的博客

2019-07-04
#github
hexo

前言

搭建博客是方便自己总结记录自己学习到的知识，方便自己以后忘了的时候查阅。

开启github pages

首先我们先注册一个github账号，登录后新建一个仓库

新建仓库

然后填写仓库的名称和描述，这里一定要注意仓库名称的格式：用的用户名.github.io，填完点击创建仓库即可

仓库格式

然后点击settings设置

setting设置

然后找到github pages，点击choose a theme选择主题

选择主题

主题设置

hexo环境搭建

使用hexo来丰富我们的博客主题

安装git

安装nodejs

安装完成后，创建一个存放hexo组建的目录，我这里使用blog目录为例，这个目录就是之后操作博客的目录

打开git bash，进入blog目录，执行npm install hexo -g,安装hexo

安装hexo
